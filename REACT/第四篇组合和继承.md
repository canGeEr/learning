# 组合和继承

## 包含关系(插槽 洞的预留)
props是可以传递组件的本身就是一个对象
```javascript
//默认 props.children = 组件中间值 列如

<FrameSet
  left = {<Bar />}
  bottom = {<Foot />}
>
</FrameSet>

//Frameset.jsx文件
function FrameSet(props) {

  return (
    <div> 
      <header>{props.left}</header>
      {props.bottom}
    </div>
  )
}
```
