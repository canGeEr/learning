# 地址和引用

## String 不用 == 比较
1. 字符串是不可变的数据
2. 所谓的字符串在公共的存储池共享，**共享的只有字符串字面量**
```java
  "aaa".subtring(0,1) != "a" //false
  //推荐equals , equalsIgnoreCase
```
3. 拼接 值得注意的几个点
(1) Boolean 转换为字符串 直接转换 true => "true"
(2) Object ... 包名 + 类名 + @ + 地址值(这个是猜测但是看起来像)
(3) System.out.print 在传入对象是会调用对象的toString(如果被重写那啥)
