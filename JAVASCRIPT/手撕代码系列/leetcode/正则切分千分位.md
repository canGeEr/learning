## 正则实现

```javascript
let str = "12345678.1212";

function splitByTh(str) {
  return str.replace(/(\d)(?=(\d{3})+$)/g, "$1,");
}

function splitByTh(str) {
  return str.replace(/(\d)(?=(\d{3})+\.)/g, "$1,");
}
```

?= 是不记录匹配的，匹配到的依然可以作为新的匹配起点
